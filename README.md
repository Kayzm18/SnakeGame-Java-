SnakeGame-Java

A simple Snake game implemented in Java.

Description

This project is a basic version of the classic Snake game written in Java. The game uses standard Java libraries to provide a graphical interface and keyboard controls for gameplay.

Requirements

Java Development Kit (JDK) installed (Java 8 or newer)

How to Run

Clone the repository

git clone https://github.com/Kayzm18/SnakeGame-Java-.git
cd SnakeGame-Java-


Compile the code

javac path/to/your/*.java


Run the game

java MainClassName


Replace MainClassName with the name of the class containing public static void main.

Controls

Use arrow keys to control the snake’s direction.
